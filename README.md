## Longshot 到底是个啥

Longshot 是开发者 chitaner 做的一款 macOS 截图贴图工具，对标的就是 Snipaste 和 Xnip 那一拨老牌选手。它把截图、标注、贴图钉图、滚动长截图、OCR 文字识别、屏幕测量、轻量录屏、取色这些活儿，全塞进一个小巧的应用里，每一样都做得挺扎实。

我自己用下来最顺手的是这几个：

**滚动长截图**这块，Longshot 在 Mac 平台上算得上顶尖水准。支持竖向、横向、全景三种方向滚动，拼接算法会自动处理边缘和背景，不用你手动对齐。截长网页、长聊天记录、长文档，一滚就完事。

**贴图钉图**是把任意图片、文本、甚至 HTML 直接钉在桌面最上层，支持滚轮缩放、旋转镜像。写文档对照资料、做设计参考图，这个功能比双屏还方便。

**OCR 文字识别**走的是离线算法，框选屏幕区域就能把图片、视频、文档里的文字抠出来，直接进剪贴板，粘贴到哪儿都行。专业版还能识别二维码和条形码，识别完甚至能语音播报。

**窗口组合截图**也巧妙——智能识别窗口快速截取，按住 Shift 还能多窗口组合截图，做教程截图特别省事。
